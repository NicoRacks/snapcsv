# SnapCSV — Table to CSV Exporter

A Chrome extension that exports any webpage HTML table to CSV in one click.

## Features
- Detects all HTML tables on the current page
- Preview before you export
- Smart header detection (handles section headers, headerless tables)
- RFC 4180 compliant CSV output
- 10 free exports/day · Unlimited with Pro

## Privacy
SnapCSV does not collect or transmit any personal data. All processing happens locally in your browser. See the full privacy policy at https://nicoraks.github.io/snapcsv/
